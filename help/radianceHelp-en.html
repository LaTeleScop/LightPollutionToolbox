Contribution to the luminous halo (Radiance)

The objective of this indicator is to classify the lit areas according to their contribution to the luminous halo, based on their level of radiance. It is not a question here of directly modeling the luminous halo but of classifying the geographical units (grid) according to the emissions upwards, captured by the satellite and contributing to the formation of the luminous halo.

Input parameters

  • Extent zone : Extent of the study area used to delimit the calculated indicator.
  • Satellite Image: Night Satellite Image in raster format (image with 3 RGB bands, or with a single band which corresponds to the total radiance).
  • Choice of grid: Grid on which the indicator will be calculated, it can be imported or created.
    • Import a grid: Grid vector layer already created.
    • Create a grid :
      • Diameter : Grid diameter, meters
      • Grid type: Type of the grid to create.

Output data

  • Statistics Radiance Output : Vector layer representing the average radiance calculated by grid with a classification by quantile in 5 classes from the lowest values ​​to the highest, with also a class representing the grid without radiance.
  • Raster total Radiance Output : Raster layer with the total radiance per pixel (only if the input satellite image has at least 3 spectral bands).
    • Save to temporary file if not filled.
