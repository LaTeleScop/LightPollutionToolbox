Visibility Light Source

The light source visibility indicator allows to evaluate at each grid, the number of light sources visible by an observatur at a given height.
It requires several data calculated in tab 1 and 2, to then calculate the results in tab 3.

DSM

DSM (Digital Surface Model) is calculate adding to the DEM, buildings with its height and possibly the vegetation (optional).

Input parameters

  • Extent zone : Extent of the study area used to delimit the calculated indicator.
  • DEM : Digital Elevation Model in raster format.
  • Radius for visibility : Buffer distance around extent zone (similar to viewshed visibility radius).
  • Buildings : Vector layer of the building from the Topo DB.
  • Height Buildings field : Field representing the height in the building layer.
  • Vegetation : Vegetation vector layer from the Topo DB (optional). To avoid having vegetation overlapping buildings, a difference is applied between the vegetation and building slayer with a buffer of 5 meters.
  • Height Vegetation field : Field representing the height in the vegetation layer (optional), vegetation with null height is not taken into account.
  • Height value applied to all vegetation if no height field chosen.

Output data

  • DSM Raster : Digital Surface Model raster layer with buildings and vegetation if a layer has been added.
  • Raster buildings vegetation : Raster layer with pixel values ​​the height of buildings and the vegetation if a layer has been added.
