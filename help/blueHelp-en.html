Emission in the blue part of the spectrum

The purpose of this indicator is to classify light sources according to the "red emissions/blue emissions" ratio, calculated from nighttime satellite images, and to identify which areas and which uses are most affected by blue light.

Input parameters

  • Extent zone : Extent of the study area used to delimit the calculated indicator.
  • Satellite Image: Night Satellite Image in raster format.
  • Choice of grid: Grid on which the indicator will be calculated, it can be imported or created.
    • Import a grid: Grid vector layer already created.
    • Create a grid :
      • Diameter : Grid diameter, meters
      • Grid type: Type of the grid to create.

Output data

  • Statistics Blue Emmission Output : Vector layer representing the indicator calculated by grid with a classification by quantile in 5 classes from the lowest values ​​to the highest, with also a class representing the grids without blue emission.
    • Save to temporary file if not filled.
